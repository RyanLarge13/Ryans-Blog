Dynamically Typed Language

Javascript is a dynamically typed language. This is extremely important to understand so let me make it clear with examples of what I mean by a dynamically typed language and how it works. There are two different ways a language can determine its types of variables:

  • Dynamically
  • Statically

What are you looking at here? Lets break it apart. in this short code snippet basically you are going to need to read it backwards. I am using C in this example, and many languages such as JavaScript, C++ and more use the same conventions as this where a ";" is used at the end of a statement for what they call a statement delimiter, helping a compiler determine the end of one statement of code to the next.


- Now, just to the right of our semicolon we see a value. Here we put the value 24.


- To the left of this we have our assignment operator "="


- And to the left of that we name our value. We are going to name the value of 24 and its address in memory as the text "myNumber"


- Finally we have the part of this statement that actually matters. "int"..
